ในปัจจุบัน คอมพิวเตอร์ เข้ามามีบทบาทที่สำคัญต่อสังคมของมนุษย์เรา คอมพิวเตอร์มีมาตั้งแต่สมัยโบราณเมื่อมนุษย์ต้องการ เครื่องช่วยในการคำนวนซึ่งคอมพิวเตอร์ เริ่มต้นจากในสมัยโบราณผู้คนต้องการสร้างเครื่องที่ช่วยในการคำนวน


เมื่อ4-5พันปีก่อน มนุษย์ในสมัยนั้นประดิษฐ์ ลูกคิด มาเพื่อช่วยคำนวน


ในยุคกลางไม่ใช่ทุกคนที่ได้เข้าเรียนหนังสือ จึงมีคนประดิษฐ์ Napier’s bone เพื่อช่วยให้ผู้คน x ÷ เลขได้ง่ายขึ้น


ต่อมาในปี 1642 blase pascal ได้สร้าง Pascaline เพื่อช่วยในการบวกและ ลบ ซึ่งจะมีแป้นคล้ายๆโทรศัพท์สมัยก่อนในการใส่ตัวเลข และมีช่องสำหรับแสดงผล แต่เครื่องนี้ทำงานได้ไม่ค่อยเสถียรเท่าใหร่ และมีราคาค่อนข้างแพง [รูป Pascaline]

stepped reckoner สร้างโดย Gottfried heinnie ในปี1672 ซึ่งเครื่องนี้สามารถ + - x ÷ ได้ มันมีที่สำหรับหมุนเพื่อว่า เมื่อเราตั้งเลขไว้และหมุนหนึ่งรอบ= x1 แต่ก็ยังคงไม่ค่อยเสถียร


arithmometer สร้างในปี 1820 มันคล้ายๆกับ stepped reckoner แต่ค่อนข้างเสถียรกวว่า เป็นเครื่องคำนวณที่ประสบความสำเร็จในเชิงพาณิชย์เครื่องแรกที่สามารถทำการบวก ลบ คูณ หารได้ และมีการผลิตและขายจนถึงศตวรรษที่ 20


Charles Babbage คนส่วนมากในสมัยนี้ยกให้เขาเป็นบิดาแห่งคอมพิวเตอร์ เขาเป็นพ่อค้า และสร้างเครื่อง ที่สามารถคำนวน polynomial function ชื่อว่าDiffenece Engine ให้กับทหาร แต่สร้างไม่เสร็จ และใน 1991 ได้มีมหาวิทยาลัย นำแบบที่เขาทำมาสร้างใหม่ ในปี 1837เขาออกแบบเครื่อง ที่เขาเรียกว่า Analytical engine ซึ่งเป็นเครื่องcomputer อเนกประสงค์ และมีหลักการทำงานคล้ายกับcomputerสมัยนี้ มีinput เป็นการใส่บัตรเจาะรู มีส่วนที่คำนวน มีmemory และ output พิมพ์ลงกระดาษ แต่เครื่องนี้ไม่ได้ถูกสร้างขึ้น


ada lovelace เป็นนักคณิตศาสตร์ และเป็นเพื่อนกับ Charles Babbage เธอได้มาเห็น เครื่อง Diffenece Engine และแบบของ Analytical Engine และคิดว่าเครื่อง Analytical engine เป็นมากว่าแค่เครื่องคำนวน และเธอ เขียนตัวอย่างการเขียนprogram ลงในเครื่อง Analytical โดยใช้ แผ่นเจาะรู ซึ่งถือว่าเธอเป็นprogramer คนแรกของโลก


หลังจากนั้นมีการสร้างcomputer มาเรื่อยๆ ในช่วงศตวรรษที่ 20 computer ส่วนมากจะเป็นแบบกึ่ง machanical กับ electronic และสามารถคำนวนได้เฉพราะประเภท


ในปี1946 กองทัพอเมริกา ร่วมกับ university of pennsylvania ได้สร้างเครื่อง ENIAC ย่อมาจาก Electronic Numerical Integrator and Computer ซึ่งเป็น computer เครื่องแรก ที่เป็นทั้ง electronic, digital, general purpose, และ programmable ซึ่งพัฒนาในช่วง สงครามโลกครั้งที่ 2 และเป็นความลับ เครื่องENIAC input ด้วยบัตรเจาะรู มีหน้าตาเป็นตู้ขนาดใหญ่ หลายๆตู้ต่อกัน ใช้หลอดสูญญกาศ และไฟฟัา โดยมีการพัฒนาเรื่อยๆ จนตอนหลัง ใช้หลอดสูญญกาศถึง 18,00 อัน มี 140ตู้ น้ำหนัก 27 ตัน ขนาด 170 m^2 กินไฟ 150 kw.


คอมพิวเตอร์ digital ทำงานโดยใช้เลขฐานสองในการคำนวน ซึ่งวิธีการของเลขฐานสองนั้นจะใช้switch ในการประมวลผล โดยเลข 1 แทน switch เปิด และเลข 0 แทน switch ปิด


ในช่วงแรกๆ computer ใช้หลอดสูญญกาศ เป็นswitch แต่ต่อมาใช้เป็น transistor


switch มีสองประเภท แบบแรกคือ ต้องมีไฟเข้าทั้งสองขา ถึงจะปล่อยไฟผ่าน


ตัวอย่างการทำงานของ switch สามารถทำเป็นเครื่องคิดเลขแบบง่ายๆ


แบบที่สองคือ ต้องมี ไฟเข้าเข้าแค่ขาใดขาหนึ่ง ถึงจะปล่อยไฟผ่าน


ในปี 1957 มีการใช้ keybord แทนบัตรเจาะรู


Output สมัยก่อนเป็น หลอดไฟ ต่อมาใช้เป็นจอแสดงผล และพิมพ์คำสั่งในการใช้งาน (command line intorface)


ในสมัยก่อนคอมพิวเตอร์แต่ละรุ่น มี software เป็นของตัวเองทำให้เมื่อ ผู้ใช้เปลี่ยน รุ่นต้องเรียนรู้การใช้งานใหม่ ปี 1964 บริษัท IBM เปิดตัว IBM system/360 ซึ่งเป็นคอมพิวเตอร์main frame เครื่องแรก ที่สามารถใช้ อุปกรณ์ และ software ร่วมกับ รุ่นอื่นๆได้


Apple เป็นหนึ่งในบริษัท ที่มีชื่อเสียง โดยเริ่มจาก เพื่อนสองคนชื่อ Steve Woznlak และ Steve Jobs Apple I ปี 1976 เป็น คอมพิวเตอร์เครื่องแรกของ บริษัท Apple


ในปี 1981 IBM PC เป็นคอมพิวเตอร์ส่วนบุคคล ที่ได้รับความนิยม อย่างมาก


Osborne 1 ปี1981 เป็นคอมพิวเตอร์พกพา เครื่องแรกที่ประสบความสำเร็จเชิงพาณิชย์


Apple Lisa เป็นDesktop computer เปิดตัววันที่19 มกราคม ค.ศ.1983 เป็นคอมพิวเตอร์ส่วนบุคลและใช้ระบบ GUI ซึ่งการใช้ GUI ทีมงานของ Apple ได้เอาแนวคิดนี้มาจากเครื่อง Xerox Alto ซึ่งคอมพิวเตอร์เครื่องแรก ที่ แสดงผลเป็น graphical user interface เปิดตัวเมื่อวันที่ 1 มีนาคม 1973 ซึ่งถึงแม้จะเป็นpersonal computer แต่ก็มีราคาค่อนข้างสูง ทำให้มีคนที่ได้ใช้น้อย


คอมพิวเตอร์ยุคแรกๆ ไม่มี memory ในตัว เมื่อใช้ต้องใส่แผ่น floppy disk โดยจะมีสองแผ่น แผ่นแรก เพื่อเป็นprogram ที่เราจะใช้ และแผ่นที่สอง เป็นข้อมูลที่เราจะให้คอมพิวเตอร์คิด ต่อมามีคอมพิวเตอร์เป็นแบบ storedprogram คือคอมพิวเตอร์จะเก็บ program ใว้ในmemory ของตัวเอง


PowerBook เป็นคอมพิวเตอร์ที่พกพา จาก Apple.inc ซึ่ง ออกวางขายในช่วงปี1991 ถึง2006 ซึ่งระหว่างนั้นได้ผ่านการแก้ไขและออกแบบใหม่หลายครั้ง ก่อนที่Powerbook จะถูกแทนที่ด้วย macbook pro ในปี 2006
